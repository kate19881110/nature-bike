# 山海戀Bike租車後台系統
* 典型的後台系統，主題關於會員租腳踏車資料、購買周邊產品銷售狀況。
* 根據數據推出行銷方案

# 專案架構
1. 前端框架: React 16.0
2. 設計套件: Ant Design
3. Echart
4. json server
5. 測試框架
